## Getting Started

### Prerequisites

- Go 1.23.2 or higher

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/backup_slack.git
   ```

2. Install dependencies
   ```bash
   go mod tidy
   ```

3. Copy the example environment file
   ```bash
   cp .env.example .env
   ```

4. Run the application
   ```bash
   go run cmd/backup_slack/main.go
   ```
